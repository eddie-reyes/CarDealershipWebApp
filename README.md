# Vehicle Marketplace Applet (JSP)

A Java web applet-style marketplace application where users can browse and purchase **cars**, **motorcycles**, and **boats**. The UI is rendered using **JSP (JavaServer Pages)** to dynamically generate HTML based on server-side state.

---

## Features

- **Vehicle catalog** by category: Cars, Motorcycles, Boats
- **Dynamic JSP pages** that reflect server state (inventory, cart, orders)
- **Product details** view (price, specs, availability)
- **Shopping cart** (add/remove/update quantities)
- **Checkout flow** (create an order and reduce inventory)
- Basic server-side validation (inventory checks, required fields)

---

## Tech Stack

- **Java** (Servlet/JSP)
- **JSP** for server-side HTML rendering
- **Servlets** for request routing / controller logic
- **JSTL** (recommended) for cleaner JSP logic
- **Apache Tomcat** (or any Servlet container)
- **HTML/CSS** (optionally Bootstrap/Tailwind via CDN)
